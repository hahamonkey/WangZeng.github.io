CSS实现水平垂直居中

发表于 2019-03-07 | 更新于 2019-03-10

作为一名前端工程师，处理居中问题是。实现 CSS 居中的方式有很多种，各有利弊，这里我总结一下我遇到过或者用过的方案。

水平居中

内联元素水平居中

利用 text-align: center 可以实现在块级元素内部的内联元素水平居中。这种方法对内联元素inline，内联块 inline-block ，内联表 inline-table，inline-flex 元素水平居中均有效(即行内或类行内元素)

text-align: center

块级元素水平居中
